## LIMIT OF SEQUENCE
  การหาลิมิตของลำดับว่าเป็น ลู่เข้า หรือลู่ออก

- ถ้าลู่เข้า เข้าสู่เลขอะไร
- ถ้าลู่ออก infinity

## MONOTONE SEQUENCE

Increasing

```
a1< a2 < a3 ... < an <...
```
Decreasing
```
a1 > a2 > a3 ... > an >...


```

มี 3 วิธีในการหาว่า มันเพิ่มหรือลด

1. DIFFERENCE
2. Ratio
3. DERIVATIVE
